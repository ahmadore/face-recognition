What is this about?
This application is a platform for report missing person's case, so that others can search for found persons.
Why Bother?
  • A lot of the time when missing person's are found, communication is difficult, therefore making it imposible to locate/contact their relatives.
  • In worst cases, they are found dead in cases of accident and other tragedies.
  • In situations like this, if the person has been reported missing prior, the reporters contact details are obtained, thereby making it easy to locate/contact them.
